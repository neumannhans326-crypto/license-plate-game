# License Plate Game

A game for car rides: kids spot license plates of passing cars and earn 
points for each one — the rarer the plate, the more points (based on 
official vehicle registration statistics). Each plate only counts once 
per trip. After every catch, you get a quick fun fact about the 
city/region the plate belongs to.

## Features
- Simple name-only registration, no account needed
- Shared scoreboard for all players
- Points awarded based on plate rarity
- Prevents duplicate scoring of the same plate
- Quick info + Wikipedia link after every catch
